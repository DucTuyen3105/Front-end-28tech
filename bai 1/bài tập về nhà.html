1. (a+b)2 = a2 + 2ab + b2

2. (a-b)2 = a2 - 2ab + b2

3. (a2- b2) = (a - b) * (a + b)

4. (a + b)3 = a3 + 3a2b + 3ab2 + b2

5. (a - b)3 = a3 - 3a2b + 3ab2 - b2

6. (a3 + b3) = (a + b)(a2 - ab + b2)
